## Run Locally

Clone the project

```bash
  git clone https://github.com/CrossStack-Q/Go_Backend_2024
```

Go to the project directory

```bash
  cd Go_Backend_2024/Projects/src/go-chat-react/frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
